# Контракт Экспо — лендинг

Одностраничный сайт программы **«Контракт Экспо»** — деловых B2B-встреч между экспонентами выставки и стратегическими заказчиками отрасли — в рамках выставки **«Агропродмаш 2026»**, Экспоцентр, Москва, **30–31 октября 2026** (дата уточняется).

Формат работает по технологии **Центра Закупок Сетей™**, но переговоры проходят не в отдельной зоне, а **прямо на стендах экспонентов**: закупщик идёт по персональному маршруту от стенда к стенду.

## Возможности

- Полностью статический сайт (HTML + CSS + ванильный JS), без сборки и зависимостей.
- **3 языка** с переключателем: русский, китайский, английский (`js/i18n.js`).
- Переключатель роли **поставщик / закупщик**.
- Блоки: шапка, о программе и платформе, результаты формата, «для кого», категории (7), как проходит + **схема маршрута по стендам**, галерея, отзывы, контакты, футер.
- Адаптивная вёрстка, доступность (фокус, `prefers-reduced-motion`).

## Структура

```
index.html            — разметка всех секций (в т.ч. SVG-схема стендов с маршрутом)
css/styles.css        — стили (палитра: зелёный + навы + голубой)
js/i18n.js            — словарь контента на 3 языках (RU / ZH / EN)
js/main.js            — переключение языков/ролей, рендер списков, scroll-reveal
img/                  — фото (hero, галерея, портреты контактов)
build_standalone.py   — сборка одного самодостаточного HTML (CSS+JS+фото внутри)
```

## Запуск

Локально достаточно открыть `index.html` через любой статический сервер, например:

```bash
python -m http.server 4321
# затем открыть http://localhost:4321
```

## Единый файл для отправки

```bash
python build_standalone.py
```

Соберёт один HTML со встроенными стилями, скриптами и фотографиями (открывается двойным кликом без интернета).
